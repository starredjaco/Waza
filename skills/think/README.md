# /think

Challenge assumptions and define scope before writing a single line of code.

## Usage

Type `/think` in Claude Code, or describe a new feature or product idea. Claude will run a structured brainstorm before any implementation begins.

## Core Method

- Step 0: Premise Challenge -- is this the right problem to solve?
- Scope Mode selection: EXPANSION / SELECTIVE / HOLD / REDUCTION
- 8 CEO Thinking Patterns (Jobs, Bezos, Musk, Grove, Buffett, Bezos again, Rams, Completeness)
- Design Output: what we're building, what we're NOT building, key decisions
- HARD GATE: no code until design is explicitly approved

## Install

Single skill:
```bash
claude plugin marketplace add tw93/waza
claude plugin install think@waza
```

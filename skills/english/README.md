# /english

Correct grammar and sharpen phrasing for non-native English speakers working in international tech.

## Usage

Type `/english` followed by the text you want coached. Claude will run a full session on it.

## Core Method

- Corrected version: full rewrite with natural, professional phrasing
- Changes list: one line per change, pattern name first, then the fix
- One pattern to practice: the most important recurring mistake, with a name, a rule, and two example pairs

## What it targets

- Grammar accuracy: articles, prepositions, tense, subject-verb agreement
- Natural phrasing: word choice that sounds like a native speaker, not a textbook
- Professional register: emails, Slack messages, PR comments, meeting English
- Confidence markers: direct opinions, clear pushback, clean closings

## What it does not do

- Does not replace your vocabulary with more sophisticated words just to sound impressive
- Does not correct intentional informal tone
- Does not explain the same pattern twice in one session

## Install

```bash
npx skills add tw93/Waza -a claude-code -s english -y
```
